# CS 499 ePortfolio Website

This is a ready-to-upload GitHub Pages site for the CS 499 capstone.

## What you should do now

1. Open your existing ePortfolio repository on GitHub.
2. Choose **Add file → Upload files**.
3. Upload `index.html`, `styles.css`, the `assets` folder, and the `downloads` folder.
4. Commit to the `main` branch.
5. Confirm **Settings → Pages** is set to `main` and `/ (root)`.
6. Open the published URL after GitHub finishes deploying.
