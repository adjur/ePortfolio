Selected artifact

Weight Tracking App

The Weight Tracking App was created in CS 360: Mobile Architecture and Programming. It allows users to create an account, log daily weight, view and manage previous records, and receive an SMS notification when a fixed goal is reached.

Why this artifact was selected

The project contains interface design, Java application logic, an SQLite database, validation, authentication, permissions, and data display. It can therefore demonstrate growth in all three required capstone categories.

Software engineeringUI, activities, navigation, preferences, maintainability
AlgorithmsSorting, comparisons, trends, progress calculations
DatabasesSQLite schema, CRUD, authentication, persistent data

Milestone One

Code Review

The code review explains current functionality, evaluates limitations, and introduces the planned enhancements.

  • Current functions: login, weight entry, history management, SQLite storage, and SMS permission handling.
  • Main findings: basic UI, fixed goal and phone number, plain-text passwords, limited analytics, and no user personalization.
  • Direction: themes, preferences, improved design, trend analysis, and stronger database structure.
